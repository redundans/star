# Star

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/redundans/star.svg)](https://packagist.org/packages/redundans/star) [![Total Downloads](https://img.shields.io/packagist/dt/redundans/star.svg)](https://packagist.org/packages/redundans/star)

A [Flarum](https://flarum.org) extension. Star a post in flarum.

## Installation

Install with composer:

```sh
composer require redundans/star:"*"
```

## Updating

```sh
composer update redundans/star:"*"
php flarum migrate
php flarum cache:clear
```
